# 旅ノート（TabiNote）プライバシーポリシー

**最終更新日**: 2026年6月18日

旅ノート（TabiNote）（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリが収集、使用、保存する情報について説明します。

本アプリは個人により開発・運営されています。収集した情報の取り扱いには細心の注意を払いますが、個人運営の特性上、大規模サービスと同程度のセキュリティ体制を保証するものではありません。

---

## 1. 収集する情報

本アプリは、以下の情報を収集・保存します：

### 1.1 アカウント情報
- **メールアドレス**: アカウント作成・認証に使用します。「Appleでサインイン」をご利用の場合、Apple の設定によりメールアドレスを非公開（Apple のプライベートリレーアドレス）にすることができます。その場合、本アプリには Apple が発行する転送用アドレスが連携されます
- **パスワード**: メールアドレスで登録した場合の認証に使用します（ハッシュ化されて保存され、開発者も平文で確認できません）。「Appleでサインイン」のみでご利用の場合、本アプリはパスワードを保持しません
- **ユーザー名**: ユーザーごとにユニークなID（英小文字・数字・アンダースコアで構成）
- **表示名**: プロフィールに表示される名前（2文字以上）。「Appleでサインイン」の初回のみ、Apple から提供される氏名（任意）を表示名の初期値として利用する場合があります
- **プロフィール画像**: ユーザーがアップロードしたアイコン画像

なお、本アプリは「メールアドレスとパスワードによる登録」または「Appleでサインイン」のいずれかの方法でアカウントを作成できます。「Appleでサインイン」では、Apple から本アプリへ、ユーザー固有の識別子・氏名（初回のみ・任意）・メールアドレス（または非公開のプライベートリレーアドレス）が連携されます。Apple 側で共有する情報の範囲は、サインイン時にユーザーが選択できます。

### 1.2 旅の情報
- **旅のタイトル・説明**: ユーザーが作成した旅の記録
- **旅の共同メンバー**: ユーザーが旅に追加した参加者の情報
- **日付情報**: 旅の開始日・終了日、各目的地の訪問日
- **写真・動画**: 旅の思い出としてアップロードされたメディアファイル（※ 現行のアプリでは、ユーザーが作成・投稿した旅は本サービス上で他のユーザーにも表示されます。そのURLが検索エンジン等によってクロール・インデックスされる可能性があります）
- **位置情報**:
  - 写真に含まれるGPS情報（EXIFデータ）
  - ユーザーが手動で入力した場所情報
  - 端末の位置情報（位置情報の利用を許可した場合。地図上の現在地表示や、写真への位置情報の自動設定に使用します）
- **タグ情報**: 場所タグ、カスタムタグなど

### 1.3 その他の情報
- **友達関係**: 友達申請・承認・友達一覧に関する情報
- **いいね情報**: 旅へのいいね機能の利用状況
- **通知**: アプリ内の通知一覧に表示するための情報（友達申請、旅への招待、いいね、旅の更新など）
- **通知設定**: 上記の通知種別ごとの受信 ON/OFF の設定
- **端末の通知許可状態**: お使いの端末で通知を許可した場合、バッジ等の表示に使用します（許可しなくても、アプリ内の通知一覧は利用できます）

### 1.4 任意のプロフィール属性（設定画面で入力する場合）
- **生年月日**、**性別**、**居住地（国・都市）**、**旅行スタイル**（例: 一人旅・家族旅行 等）: アカウント設定で任意入力いただく場合があります。いずれも未入力のままでも、本アプリの主要機能を利用できます
- **利用目的**: サービス品質の向上や、将来的な表示・おすすめのパーソナライズに利用する場合があります
- **公開範囲**: 設計上、他ユーザー向けの公開プロフィール画面には表示しない情報として扱います（アプリのバージョンにより公開範囲が変更される場合は、本ポリシーまたはアプリ内通知でお知らせします）

### 1.5 利用状況・端末情報（アプリの利用ログ）

本アプリは、サービスの品質向上や不具合の把握のため、アプリの利用状況に関する以下の情報を収集・保存します。これらの情報は、ご利用中のアカウント（ユーザーID）と紐づけて記録されます。

- **利用ログ（イベント情報）**: アプリの起動（セッションの開始）、表示した画面、旅の投稿などの主要な操作の種別と、その発生日時
- **端末・アプリ情報**: プラットフォーム（iOS / Android）、アプリのバージョンおよびビルド番号、OS のバージョン

利用ログには、氏名・メールアドレス・パスワードなどの情報を直接含めない方針で運用します。これらの情報は、本アプリの改善や利用状況の把握を目的として、運営者による統計・分析に利用します。広告目的での利用や、第三者への販売は行いません。

## 2. 情報の使用方法

収集した情報は、以下の目的で使用します：

### 2.1 サービスの提供
- アカウント認証とセッション管理
- 旅の記録の保存・表示
- 友達機能・旅の共同メンバー機能の提供
- いいね、実績（訪問した都道府県・国など）の表示
- 地図上での旅の表示、ユーザー検索・@メンションによる旅の絞り込み
- アプリ内通知の配信および通知設定の反映

### 2.2 サービスの改善
- アプリの動作確認と不具合の修正
- パフォーマンスの最適化
- 新機能の開発
- アプリの利用状況の分析・統計（どの機能・画面が利用されているか等の把握。1.5 の利用ログ・端末情報を含みます）

### 2.3 ユーザーサポート
- お問い合わせへの対応
- アカウント管理

## 3. 第三者サービス

本アプリは、以下の第三者サービスを使用しています：

### 3.1 Supabase
- **用途**: 認証、データベース（ユーザー情報・旅の記録・通知など）
- **提供元**: Supabase Inc.
- **プライバシーポリシー**: https://supabase.com/privacy
- **保存場所**: データはSupabaseのサーバーに保存されます

### 3.2 Cloudflare R2
- **用途**: メディアファイル（写真・動画・プロフィール画像など）の保存
- **提供元**: Cloudflare, Inc.
- **プライバシーポリシー**: https://www.cloudflare.com/privacypolicy/
- **保存場所**: データはCloudflare R2に保存されます

### 3.3 Sign in with Apple（Appleでサインイン）
- **用途**: 認証（ユーザーが「Appleでサインイン」を選択した場合のログイン・新規登録）
- **提供元**: Apple Inc.
- **連携される情報**: ユーザー固有の識別子、氏名（初回のみ・任意）、メールアドレス（または Apple のプライベートリレーアドレス）
- **プライバシーポリシー**: https://www.apple.com/legal/privacy/
- **補足**: メールアドレスを非公開にした場合、本アプリが受け取るのは Apple が発行する非公開の転送用アドレスであり、実際のメールアドレスは本アプリには共有されません。Apple アカウントでの連携状況の確認・解除は、お使いの端末の「設定 →（ユーザー名）→ サインインとセキュリティ →『Appleでサインイン』」から行えます

## 4. データの保存とセキュリティ

### 4.1 データの保存期間・保存容量
- **保存期間**: アカウントが有効な限り、原則としてデータはクラウド上に保存されます
- **保存容量の上限**: 運営上の必要（サービス継続、セキュリティ、インフラコストの管理等）により、ユーザーごとの保存容量の上限を設けたり、長期間利用がないアカウントのデータを削除したりする場合があります。これらの条件は、[利用規約](https://kazutax.github.io/tabiNote-legal/terms-of-service.html)および本サービス上の表示で示します
- **使用量の表示**: アプリのマイページ等では、おおよそのクラウド保存使用量を表示する場合があります。表示値は、アップロード時に記録したサイズの合計等に基づくもので、クラウドストレージ上の実使用量と完全に一致しない場合があります
- **アカウント削除時**: 関連するデータも削除されます
- **メディアの削除**: 写真・動画などのメディアは、アカウント削除やユーザーによる削除に連動して削除キューに登録され、サーバー側で一定の間隔でバッチ処理により削除されることがあります。削除完了までに時間差が生じる場合があり、即時の消去を保証するものではありません

### 4.2 セキュリティ対策
- パスワードはハッシュ化されて保存されます
- HTTPS通信により、データ転送を暗号化しています
- アクセストークンを使用した認証を行っています

## 5. ユーザーの権利

ユーザーは以下の権利を有します：

### 5.1 データへのアクセス
- アプリ内の設定画面から、自分のプロフィール情報を確認・編集できます

### 5.2 データの削除
- アプリ内の設定画面から、アカウントを削除できます
- アカウント削除により、すべての個人情報と旅の記録が削除されます

### 5.3 位置情報の管理
- 写真のGPS情報は、写真のEXIFデータから自動的に取得される場合があります
- 端末の位置情報は、OSの設定およびアプリ利用時の許可ダイアログで管理できます
- 位置情報の共有を希望しない場合は、写真をアップロードする前に位置情報を削除するなどの方法をご検討ください

### 5.4 通知の管理
- アプリ内の設定画面から、通知種別ごとの受信 ON/OFF を変更できます
- 端末の通知（バッジ等）は、端末の「設定」アプリから管理できます

---

## 6. データの共有

本アプリは、以下の場合を除き、ユーザーの個人情報を第三者と共有しません：

- 法的義務に基づく開示が必要な場合
- ユーザーの明示的な同意がある場合
- 本プライバシーポリシーに記載された第三者サービスへの提供
- ユーザーが投稿した旅の内容を、本サービスの他の利用者が閲覧する場合

## 7. 未成年のご利用について

本アプリは、特定の年齢層を主要な対象とするものではありません。未成年の方がご利用になる場合は、保護者の方の同意のもとでご使用ください。本アプリは、未成年の方の個人情報を意図的に収集することはありません。

## 8. プライバシーポリシーの変更

本プライバシーポリシーは、予告なく変更される場合があります。重要な変更がある場合は、アプリ内で通知します。変更後もアプリを継続して使用することで、変更されたプライバシーポリシーに同意したものとみなされます。
